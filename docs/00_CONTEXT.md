# 00 Context

## One Line

HeatHub is a European emergency weather search platform that helps users find where to buy or access urgent cooling products and related services.

## Core Principle

HeatHub provides information only.

It does not sell products, process payments, create orders, manage stock, or handle delivery.

## Main Use Case

A user is experiencing extreme heat and needs a fast answer:

- What can I buy?
- Where can I buy it?
- Can it be delivered today?
- Is installation available?
- Which merchant is nearby?
- Which result is the fastest or cheapest?

## Initial Geography

Start with Europe.

Suggested first markets:

- France
- Germany
- Spain
- Italy
- Netherlands

## Initial Weather Focus

MVP focuses on extreme heat.

Future versions may expand to:

- Power outage
- Flood
- Storm
- Cold wave

## Business Model

Potential revenue sources:

- Affiliate links
- Sponsored listings
- Merchant subscription
- CPC advertising
- Local service provider leads

No revenue should depend on HeatHub handling checkout.
