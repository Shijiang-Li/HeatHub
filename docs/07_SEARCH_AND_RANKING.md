# 07 Search And Ranking

## Search Inputs

- Keyword
- City
- Country
- Current location
- Category
- Delivery requirement
- Installation requirement
- Price range

## Filters

- Category
- Price
- Delivery speed
- Pickup available
- Installation available
- Distance
- Merchant rating
- Data freshness

## Sort Options

- Recommended
- Fastest delivery
- Lowest price
- Nearest
- Highest rated
- Recently updated

## Recommended Ranking Formula

Recommended score should be explainable.

Suggested signals:

- Delivery speed
- Distance
- Price
- Rating
- Installation availability
- Data freshness
- Weather urgency

Avoid black-box ranking in the MVP.

## Result Explanation

Each result can show short reasons:

- "Fastest delivery"
- "Nearest pickup"
- "Installation available"
- "Recently updated"
- "Lowest price in this search"
