# 09 Tech Stack

## Application

- Next.js App Router
- React
- TypeScript
- Tailwind CSS

## Data

- PostgreSQL
- Prisma
- Supabase

## Search

MVP:

- PostgreSQL full-text search
- Indexed city, category, merchant, and product fields

Future:

- Meilisearch
- Typesense
- OpenSearch

## Maps

- MapLibre
- OpenStreetMap-compatible tiles

## Weather

- OpenWeather or equivalent official provider

## Analytics

- PostHog or privacy-friendly alternative

Track:

- Search events
- Result clicks
- Outbound clicks
- City/category demand

## Deployment

- Vercel for frontend and API routes
- Supabase for database
- Cloudflare for CDN and caching

## Monitoring

- Sentry or equivalent
- Basic uptime monitoring
