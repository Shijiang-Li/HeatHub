# 11 Security Privacy Compliance

## Privacy

Collect the minimum data needed.

For MVP, avoid user accounts unless necessary.

Do not store precise user location longer than needed for search.

## GDPR

For Europe, prepare for:

- Cookie consent where required
- Analytics consent where required
- Privacy policy
- Data deletion request process

## Affiliate Transparency

Outbound links should be transparent.

If affiliate links are used, disclose that HeatHub may earn a commission.

## Data Rights

Each data source must record:

- Source name
- Source type
- License or terms notes
- Allowed use
- Update frequency

Do not use unauthorized data sources.

## User Trust

Show freshness labels such as:

- "Updated 15 minutes ago"
- "Updated today"
- "Data may change on merchant site"
