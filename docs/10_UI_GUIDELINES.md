# 10 UI Guidelines

## Design Direction

- European minimalist
- Fast and practical
- High readability
- Mobile-first
- Trustworthy
- No dark patterns

## Core Components

- SearchBar
- LocationInput
- CategoryTabs
- ResultCard
- ProductCard
- MerchantCard
- ServiceCard
- WeatherBanner
- MapPanel
- FilterDrawer
- SortMenu
- OutboundLinkButton

## Result Card Must Show

- Name
- Merchant or provider
- Price when available
- Distance when available
- Delivery or response estimate
- Installation availability when relevant
- Last updated time
- Clear outbound action

## Outbound Link Copy

Use clear labels:

- "Open merchant website"
- "View on official site"
- "Contact provider"

Avoid misleading labels:

- "Buy now"
- "Checkout"
- "Place order"

## Accessibility

- Use semantic buttons and links.
- Maintain visible focus states.
- Keep color contrast readable.
- Do not rely on color only for alert levels.
