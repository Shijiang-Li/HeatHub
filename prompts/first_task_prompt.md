# First Task Prompt

Create the initial HeatHub Next.js project.

Requirements:

- Next.js App Router
- TypeScript strict mode
- Tailwind CSS
- ESLint
- Basic folder structure
- Home page
- Search page placeholder
- Product result card component
- Merchant card component
- Weather banner component
- Mock seed data

Do not create ecommerce features.

Do not create cart, checkout, payment, orders, inventory, or logistics.

The first version should demonstrate:

- Search input
- Category selection
- Mock results
- Clear outbound merchant link
- Mobile-first layout
