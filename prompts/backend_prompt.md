# Backend Prompt

Build backend/API features for HeatHub.

Use:

- Next.js Route Handlers
- TypeScript
- Prisma
- PostgreSQL

APIs are mainly read-only.

Allowed API areas:

- Search
- Products
- Merchants
- Services
- Weather
- Outbound click tracking

Forbidden API areas:

- Cart
- Checkout
- Payment
- Orders
- Refunds
- Inventory management
- Logistics

All responses must be typed and predictable.
