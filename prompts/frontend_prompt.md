# Frontend Prompt

Build frontend features for HeatHub.

Use:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Server Components first

Pages should be mobile-first, fast, and SEO-friendly.

Every data view must include loading, error, empty, and success states.

Do not use ecommerce language such as cart, checkout, buy now, or place order.

Use outbound language:

- Open merchant website
- View on official site
- Contact provider
