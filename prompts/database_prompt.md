# Database Prompt

Design the database for HeatHub as an information aggregation platform.

Include:

- Merchant
- Product
- ServiceProvider
- Category
- City
- WeatherSnapshot
- SearchEvent
- OutboundClick
- DataSource

Do not include:

- Cart
- Checkout
- Payment
- Order
- Refund
- Shipment
- Warehouse

Use Prisma schema and PostgreSQL-friendly indexes for search, city, category, and outbound analytics.
