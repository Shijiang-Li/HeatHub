# Codex Instructions

You are working on HeatHub.

Read `CODEX.md` first, then read the files in `docs/`.

The most important rule:

HeatHub is not ecommerce.

Do not create cart, checkout, payment, orders, refunds, inventory, warehouse, or logistics features.

Build only:

- Search
- Aggregation
- Comparison
- Weather-aware results
- Map discovery
- Outbound merchant/provider redirects
- Affiliate or outbound click tracking

When writing code, prefer simple production-ready implementation over complex architecture.

If product behavior is unclear, ask before coding.
